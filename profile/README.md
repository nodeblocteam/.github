# NodeBloc

**NodeBloc** is a blockchain development organization focused on building practical Web3 applications, protocol tools, and on-chain products.

We build open-source blockchain solutions across DeFi, gaming, infrastructure, and real-world adoption.

---

## What We Build

- Smart contracts
- Blockchain applications
- DeFi tools
- On-chain games
- Protocol experiments
- Web3 infrastructure
- Open-source developer tools

---

## Our Mission

To build simple, secure, and useful blockchain products that solve real problems.

NodeBloc is focused on turning blockchain ideas into working applications that people can use.

---

## Tech Stack

**Languages:** Solidity, Go, Rust, TypeScript, JavaScript  
**Tools:** Foundry, Hardhat, Git, Docker
